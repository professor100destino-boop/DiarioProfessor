Pacote para adicionar Backup Completo ao app DiarioProfessor.

Arquivos:
- funcoes_backup_para_index.js: funções D.exportBackup e D.importBackup para colar no index.html.
- botoes_backup_para_viewSettings.txt: bloco visual dos botões para colar na tela Segurança.
- sw.js: service worker atualizado de diario-v3 para diario-v4.
- PASSO_A_PASSO_BACKUP.txt: instruções detalhadas.

O backup gerado será um arquivo .json com escolas, turmas, alunos, chamadas, atividades, notas e autenticação local.
